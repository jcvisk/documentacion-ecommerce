Coffee & Tea

Acerca del proyecto

Este proyecto consta de un e-commerce, el cual consta de una pagina de inicio, un catalogo de productos, fichas de productos, cesta de compras y una pasarela de pagos. Este proyecto está orientado a una cafetería por lo cual los productos serían bebidas calientes, frías y frappé, bebidas de especialidad, coctelería con café, jugos, crepas etc.

Algunas de las ventajas que proporciona este e-commerce son: mayor visibilidad, reducción de costos, disponibilidad inmediata.

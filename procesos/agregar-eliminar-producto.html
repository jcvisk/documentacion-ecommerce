Proceso de agregar o eliminar del carrito

Página de inicio

Se muestra la página de inicio.

Mostrar lista de productos

Se le muestra al usuario la lista de productos.

Seleccionar o visualizar porducto

se muestra un boton para seleccionar y otro para visualizar detalles del producto.

Visualizar porductos

Se muestran los detalles del producto.

Selecccionar porducto

Se selecciona el producto.

Agregar al acarrito

Se agrega al carrito el producto seleccionado.

Ver lista de compras

Se muestra al usuario la lista de productos agregados al carrito.

Eliminar del carrito

Para cada producto existe un boton para eliminar del carrito.

Seleccionar el producto a eliminar

Se selecciona el producto a elimnar.

Eliminar producto

Se elimina el producto seleccionado de la lista de compras.

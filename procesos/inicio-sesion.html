Proceso de inicio de sesión

¿Estas registrado?

Pregunta al usuario si ya esta registrado en el sistema, de ser así pasa al paso de ingresar usuario y contraseña, de no ser así manda al paso de registrarse.

Regístrate

Se le muestra al usuario la ventana de registro.

Ingresar correo electrónico

Se solicita al usuario ingresar un correo electrónico.

¿El correo ya está en uso?

Valida si el correo ingresado ya esta en uso, de ser así se solicita un nuevo correo electrónico, en caso contrario se procede al paso de ingresar contraseña.

Ingresar contraseña

Se solicita al usuario ingresar una contraseña.

Confirmar contraseña

Se le pide al usuario que confirme su contraseña.

¿las contraseñas coinciden?

Si las contraseñas coinciden se pasa a registar usuario, en caso de que no coincidan, se le pide nuevamente al usuario que confirme su contraseña.

Registar usuario

Se agrega un nuevo usuario a la base de datos.

Ingresar usuario y contraseña

Si el usuario esta registrado, se le pide ingresar usuario y contraseña.

¿Los datos son correctos?

Si los datos ingresados son correctos pasa al siguiente paso, en caso de que los datos no sean correctos se le pide al usuario que ingrese usuario y contraseña.

Ingresar al sistema

Se inicia una nueva sesión
